Data Types

UFO

  • Row data type: Attributes -> specific properties that can be measured or observed (ex. shape, duration)
  • Datetime: sequential quantitative datatype
  • City: categorical datatype
  • State: categorical datatype
  • Country: categorical datatype
  • Shape: categorical datatype
  • Duration: sequential quantitative datatype

Airports

  • Row data type: Items -> each airport is an individual and discrete item
  • Airport Name: categorical datatype
  • Latitude: positional datatype
  • Longitude: positional datatype

Google-Web

  • Row data type: Link -> relationship between two nodes
  • FromNodeId: item datatype
  • ToNodeId: item datatype
