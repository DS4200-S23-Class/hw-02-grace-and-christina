Data Types

UFO

  • Row Data Type: Item -> each row refers to a specific instance
  • Datetime (attribute): sequential quantitative
  • City (attribute): categorical
  • State (attribute): categorical
  • Country (attribute): categorical
  • Shape (attribute): categorical
  • Duration (attribute): sequential quantitative

Airports

  • Row Data Type: Item -> each airport is an individual and discrete item
  • Airport Name (attribute): categorical
  • Latitude: positional
  • Longitude: positional

Google-Web

  • Row Data Type: Link -> relationship between two nodes
  • FromNodeId: item
  • ToNodeId: item

Task Abstraction

High-Level Task

The high-level task of this visualization is to consume and discover. The visualization highlights the last 3 days' data points, so the user is able to determine how recent data aligns with the historical trend.

Medium-Level Task

The medium-level task of this visualization is to lookup. The highlighted data points draw the user's eye and is most likely the target, so it is clear that the location is also known.

Low-Level Task

The low-level task is to identify because the data points for the last 3 days are highlighted to stand out against the rest of the data. Therefore, the analysis is focused on just three points.

Target

The target of the visualization is outliers. It is clear that the highlighted three data points are higher than average according to the historical data plotted in grey. Therefore, it conveys that these data points are outliers.

High-Level Task

The high-level task of this visualization is to consume and present. The visualization shows the comparison of summary statistics for two different locations. There is a clear and focused purpose of the visualization.

Medium-Level Task

The medium-level task of this visualization is to browse. We know that the visualization is analyzing specific data on temperatures, so it's easy for users to understand. The visualization also contains all years, so the user can view everything and make their own conclusions, no specific data points are highlighted.

Low-Level Task

The low-level task is to compare because there are two distinct locations where the same type of information is presented, so the main motive is to compare the temperature in the two locations.

Target

The target of the visualization is similarity. Two different location's data are being compared, so the target is to determine the relationship between the two locations.
